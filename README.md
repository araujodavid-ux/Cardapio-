Projeto de Árvores.

Cardápio de Restaurante.

Projeto em Java que simula o cardápio de um restaurante usando "estrutura de árvore".
Funcionalidades
- Adicionar novas categorias e subcategorias
- Inserir itens com nome e preço
- Exibir o cardápio completo em hierarquia
Linguagem
Desenvolvido em Java, usando `ArrayList` para armazenar os filhos de cada nó.
